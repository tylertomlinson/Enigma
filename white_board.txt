Enigma - possibly super parent Class

          ENCRYPT METHOD -
            takes 3 arguments : message String, optional key and date. No key, must be generated. No date, use today's date.

              returns a hash with three keys
                1. encryption => the encrypted String
                2. key => the key used for encryption as a String
                3. date => the date used for encryption as a String in the form DDMMYY
                decrypt method

          DECRYPT METHOD -
            takes cipher text String and the Key used for encryption as arguments. Can also take date as third argument. No date, use today's date.


              returns a hash with three keys
                1. decryption => the decrypted String
                2. key => the key used for decryption as a String
                3. date => the date used for decryption as a String in the form DDMMYY
